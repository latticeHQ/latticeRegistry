# Customer Support

Triage tickets, draft responses, escalate issues, and build your knowledge base. Research customer context and turn resolved issues into self-service content.

## Skills

| Name | Type | Description |
|------|------|-------------|
| `customer-support-draft-response` | command | Draft responses to customer tickets |
| `customer-support-escalate` | command | Escalate issues to appropriate teams |
| `customer-support-kb-article` | command | Create knowledge base articles from resolved issues |
| `customer-support-research` | command | Research customer context and history |
| `customer-support-triage` | command | Triage and categorize incoming tickets |
| `customer-support-customer-research` | skill | Deep-dive into customer background and context |
| `customer-support-escalation` | skill | Manage escalation workflows and routing |
| `customer-support-knowledge-management` | skill | Organize and maintain knowledge base content |
| `customer-support-response-drafting` | skill | Draft and refine customer responses |
| `customer-support-ticket-triage` | skill | Analyze and prioritize ticket queues |
