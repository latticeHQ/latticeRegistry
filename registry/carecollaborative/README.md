# CareCollaborative

AI persona library for medical education and workplace training simulations. This namespace provides reusable modules for deploying interactive AI patients, employees, and other training personas.

## Session Types

- **Clinical**: Student-AI patient interactions for medical training
- **Workplace**: Employee-AI employee training simulations
- **Observation**: Faculty-student interactions with transcription only

## Modules

- `persona-library`: Core module for defining and managing AI personas
- `session-types`: Configuration for different evaluation session types
