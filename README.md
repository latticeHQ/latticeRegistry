## What It Does

Lattice Registry is the community-maintained collection of:

- **Templates** — Pre-configured agent environments (engineering, research, data science)
- **Terraform Modules** — Infrastructure patterns for common Lattice deployments
- **Stacks** — Domain-specific applications built on Lattice Runtime

```bash
# Apply a template from the registry
lattice templates apply engineering
```
